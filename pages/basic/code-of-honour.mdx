## Code of Honour

WE ARE, <br/>
OFFICER UNDER TRAINING, <br/>
WILL NOT TELL LIES, <br/>
CHEAT AND STEAL, <br/>
NOR BETRAY, <br/>
AND WILL NOT TOLERATE AMONG US, <br/>
THOSE WHO VIOLATE THIS CODE. <br/>

## Important

Semua warga PALAPES-Laut dikehendaki menghayati dan mengamalkan Pernyataan Arah, Nilai Teras TLDM, dan Code of Honour yang disenaraikan di perenggan 1012 hingga 1016.
