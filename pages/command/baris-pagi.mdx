## Baris Pagi

### 1) Command (MDOD): 
   - Ketua-ketua intake lapor kekuatan intake masing-masing.

### 2) MDOD Lapor kekuatan kepada OOD.

### 3) Command (MDOD): 
   - Siap sedia untuk nyanyian Lagu Samudera Raya,
   - Siap sedia untuk nyanyian Lagu UMP,
   - Siap sedia untuk nyanyian Lagu Bangga Berkhidmat.

### 4) Command (MDOD): 
   - Siap sedia Bacaan untuk Ikrar Kesatria dan Code of Honour,
   - Sedia angkat,
   - Bacaan Ikrar Kesatria dan Code of Honour selesai,
   - Sedia Turun.

### 5) Pemeriksaan baris. 

- Skuad, skuad sedia
- Buka barisan gerak
- Dengan ke kanan lurus (jarak satu depa)
- Pandang depan

| OOD  | Bagus |

minta kebenaran jalankan tugas, seterusnya kemasukan barisan.

- Tutup barisan, gerak
